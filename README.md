Ext.ux.menu.MenuBar
===================

This extension implements oldstyle menu bar widget. Nothing fancy, but
somehow it got missing from standard Ext JS widget library, so there it is.

Tested in:

* MSIE 6+
* Chrome 6+
* Firefox 3.6+
* Opera 11+
* Safari 4+

For usage, see demo application.
